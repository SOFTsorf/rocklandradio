# Rockland Radio Podcast Clone

Ein responsiver Nachbau der Podcast-Übersichtsseite von [Rockland.de](https://www.rockland.de/). Dieses Projekt dient als Übung für CSS-Layouts (Flexbox) und Dark-Mode-Design.

## Features
* **Authentisches Design:** Dunkles Farbschema und Typografie basierend auf dem Original.
* **Sticky Player:** Simulation der Player-Bar am unteren Bildschirmrand.
* **Responsive:** Optimiert für die mobile Ansicht (wie in der Rockland-App).

## Technologien
* HTML5
* CSS3 (Flexbox & Google Fonts)
