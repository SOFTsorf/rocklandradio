SiteClone Dashboard

GitHub Action-basierter Deep Crawler

1 Klon-Vorgang starten

Da Browser-Sicherheitschecks (Cloudflare) das direkte Kopieren blockieren, nutzt dieses Tool die GitHub Actions Kraft.

Gehe zum Tab Actions → Deep Asset Cloner → Run workflow.

Zu den GitHub Actions

2 Deine geklonten Seiten

Sobald der Bot fertig ist (grüner Haken in Actions), erscheinen die Seiten hier im Ordner clones/.

  • Beispiel: rocklandradio.de In Arbeit / Bereit

Hinweis: Um die Klone online zu sehen, aktiviere GitHub Pages in den Einstellungen auf den main Branch.
